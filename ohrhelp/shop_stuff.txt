This is the shop stuff editor. Here you can edit the items or heroes that can appear in this shop.

Use the arrow keys to navigate the menu. Select the Shop Thing number and prpess left or right to change which item you are editing.

You can type a name for the item or hero. This is optional, and will be automatically pre-filled when you choose an item or hero ID number.

Type can be either Item or Hero. Items are only available if the shop's "Buy" menu is enabled Heroes are only available if the shop's "Hire" menu is enabled.

Number is the ID number of the item or hero.

Stock can be "None", "Infinite" or a limited number. If your items fail to show up when you test your shops, make sure that they are not set to "None"

Buy require tag and sell require tag allow you to restrict access to certain items in this shop.

Buy set tag and sell set tag allow you to detect when certain items are purchased/sold in this shop.

You can set the price of the item in this shop. This gets auto-filled to the item's value when you choose the item ID number, but you can then edit the price just for this shop.

You can specify up to 4 items that have to be traded in to buy this item.

You can specify a sell price. This will be prefilled to half the item's value, but you can edit the sell price for just this shop.

You can specify one item to recieve as a trade-in if you sell this item.
