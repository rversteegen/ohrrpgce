Choose "Draw Tiles" if you want to edit the tiles in this tileset.

Choose "Cut Tiles" if you want to copy tiles from another tileset or extract tiles from an imported background.

Choose "Set Default Passability" if you want to define default walls for each tile in this tileset.

Choose "Define Tile Animation" to create patterns for animated tiles.

"Background color" specifies how color 0 is drawn, and affects the tileset editor only. For tilesets used map layers other than layer 0, color 0 is transparent.
